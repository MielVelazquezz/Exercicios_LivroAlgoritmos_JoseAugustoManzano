// #### Exercício 115: Gerenciamento de Notas de Alunos

// Crie um programa de computador que armazene o nome e as notas bimestrais de 20 alunos do curso de Técnicas de Programação que execute as seguintes etapas:

// a. Cadastrar os 20 registros (após o cadastro, fazer a calssificação por nome)

// c. Apresentar todos os registros e as médias dos alunos aprovados.
// d. Apresetnar apenas os registros e as médias dos alunos aprovados.
// e. Apresentar apenas os registros e as médias dos alunos reprovados.
// f. Sair do programa de cadastro.

// Situação: Aprovado

// Digite o nome para pesquisa: Maria
// Resultado da pesquisa:
// Nome: Maria
// Notas: [6.0, 5.5, 4.0, 6.5]
// Média: 5.5
// Situação: Reprovado

// **Exemplo de apresentação (opção c):**
// Todos os registros:
// - João: Média 7.5 (Aprovado)
// - Maria: Média 5.5 (Reprovado)
// ...

// Registros dos alunos aprovados:
